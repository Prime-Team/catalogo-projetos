# Catálogo de Soluções Tech 2024

Este projeto é um catálogo digital interativo de 10 nichos de mercado de alta rentabilidade em tecnologia, focado em SaaS, IA e Inovação. O design é inspirado em um livreto premium, com navegação fluida e conteúdo técnico de alto valor.

## 🚀 Funcionalidades

- **Design Premium**: Layout estilo livreto com transições suaves entre páginas
- **10 Nichos de Mercado**: Cobertura completa de SaaS, IA e tendências tecnológicas
- **Navegação Intuitiva**: Menu lateral e navegação por setas
- **Conteúdo Técnico**: Stack completa, monetização e métricas reais de mercado
- **Design Responsivo**: Otimizado para desktop e mobile
- **Modo Escuro**: Tema automático baseado no sistema

## 📋 Nichos Incluídos

1. **SaaS de Gestão Empresarial**
2. **Ferramentas de IA para Marketing**
3. **Plataformas de E-learning**
4. **Software de CRM Especializado**
5. **Soluções de Automação RPA**
6. **Plataformas de Análise de Dados**
7. **Ferramentas de Design com IA**
8. **Software de Gestão de Projetos**
9. **Plataformas de Video Marketing**
10. **Soluções de Segurança Cibernética**

## 🛠️ Stack Tecnológica

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Design**: CSS Grid, Flexbox, Animações CSS
- **Design System**: Design tokens, modo escuro, tipografia premium
- **Assets**: Imagens de alta qualidade do Unsplash

## 🎨 Design System

### Cores

- **Primária**: `#007bff` (Azul Tech)
- **Secundária**: `#6610f2` (Roxo Inovação)
- **Acento**: `#28a745` (Verde Sucesso)
- **Neutras**: Escala de cinzas para texto e fundos

### Tipografia

- **Título**: `Poppins`, 600-700
- **Corpo**: `Inter`, 300-400
- **Monospace**: `JetBrains Mono` para código

### Layout

- **Booklet**: 2 colunas em desktop, 1 coluna em mobile
- **Grid**: Sistema de grid responsivo
- **Transições**: Efeitos de virar página suaves

## 📱 Responsividade

O design se adapta perfeitamente a diferentes tamanhos de tela:

- **Desktop**: 2 colunas com layout de livreto
- **Tablet**: Transição para 1 coluna com layout otimizado
- **Mobile**: Layout de página única com navegação simplificada

## 📂 Estrutura do Projeto

```
catalogo-projetos/
├── index.html              # Página principal com todo o conteúdo
├── index.css               # Estilos globais e design system
├── README.md               # Documentação do projeto

```

## 🚀 Como Executar

1. Clone o repositório:

   ```bash
   git clone <repository-url>
   cd catalogo-projetos
   ```

2. Abra o arquivo `index.html` no seu navegador:

   ```bash
   open index.html  # macOS
   start index.html # Windows
   ```

3. Ou use um servidor local:

   ```bash
   npx serve .
   ```

## 🎨 Como Personalizar

### Adicionar Novos Nichos

1. Crie uma nova seção `<!-- NICHO XX -->` no `index.html`
2. Adicione o badge e estatísticas
3. Inclua os projetos com suas respectivas páginas
4. Atualize o sumário

### Alterar Design

Edite o arquivo `index.css` para:

- Mudar cores no `:root`
- Ajustar tipografia
- Modificar layout e transições

### Atualizar Conteúdo

Cada projeto tem sua própria seção no `index.html` com:

- Título e tipo
- Descrição
- Características principais
- Stack tecnológica
- Métricas de monetização
